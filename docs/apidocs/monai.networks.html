one_hot(labels, num_classes)[source]¶

For a tensor `labels’ of dimensions BC[D][H]W, return a tensor of dimensions BC[D][H]WN for `num_classes’ N number of classes. For every value v = labels[b,c,h,w], the value in the result at [b,c,h,w,v] will be 1 and all others 0. Note that this will include the background label, thus a binary mask should be treated as having 2 classes.

predict_segmentation(logits)[source]¶

Given the logits from a network, computing the segmentation by thresholding all values above 0 if `logits’ has one channel, or computing the argmax along the channel axis otherwise.
